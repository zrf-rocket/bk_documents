
## 蓝鲸错误码

### 一、错误码基本信息

|类别|说明|
|--|--|
| 日志码位数 | 7 位整数，前两位表示产品代号，后 5 为各产品内针对错误信息自行分配 | 
| 平台类|	以 1 开头（10-29） | 
| 公共服务	|以 5 开头（50-79） | 
| SaaS 后台	|以 3 开头（30-49） | 
| 第三方 SaaS 后台|	以 8 开头（80-98） | 
| 蓝鲸体系通用服务|	以 99 开头 | 
| -1  | 	系统繁忙 | 
| 0	 | 请求成功 |
