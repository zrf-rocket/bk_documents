## API 文档


### 一、API 概述

蓝鲸智云是一套私有化部署的软件，各产品将开放的能力，以 RESTful API 的形式接入 APIGateway，用户可以在 APIGateway 查看详细的 API 文档。当前文档提供了部分 API 文档，详细的 API 信息，以部署软件的版本为主。

特别说明：私有化部署软件 APIGateway 文档访问入口 “ {domain}/docs/apigw-api ” 。
