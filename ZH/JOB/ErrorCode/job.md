# 蓝鲸标准化错误码

## 错误码分类
1. 文件磁盘系统，格式为：59xxxxx
2. MySQL 系统，格式为：52xxxxx
3. Redis 系统，格式为：50xxxxx
4. MQ，格式为：55xxxxx
5. GSE，格式为：10xxxxx
6. CMDB，格式为：11xxxxx
7. 作业平台，格式为：12xxxxx

## 系统-通用

0=成功
