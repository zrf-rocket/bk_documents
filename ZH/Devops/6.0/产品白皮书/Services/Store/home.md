# 研发商店首页

研发商店里提供了流水线运行需要的公共资源：插件、模板、容器镜像。你可以找到适合自己项目的插件，

## 功能区介绍

![png](../../assets/service_store_home.png)

1. 通过关键字搜索当前类型的研发商店资源（当前属于什么资源类型可以在4中查看）
2. 工作台入口，可以管理自己贡献的资源
3. 当前资源列表
4. 资源类型，目前有流水线插件、流水线模板、容器镜像
5. 资源过滤器
