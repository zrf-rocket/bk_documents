# 开发一个流水线模板

## 创建模板

### 入口

流水线列表->更多->模板管理
![png](../../assets/template_entry.png)

### 模板列表

![png](../../assets/template_list.png)

1. 新增模板
2. 模板详情入口
3. 模板操作：
    - 实例化：通过模板创建流水线
    - 复制
    - 关联商店：发布到商店的快捷入口
    - 删除

### 模板编辑

![png](../../assets/template_edit.png)

1. 全局参数入口：
    - 点击后打开模板常量、流水线变量管理界面
    - 模板常量在实例中不可以修改
    - 流水线变量在示例中可以修改
2. 增加触发器
3. 通过Job、Task编排，实现模板功能逻辑
4. 模板历史版本入口，可快速回滚到历史版本
