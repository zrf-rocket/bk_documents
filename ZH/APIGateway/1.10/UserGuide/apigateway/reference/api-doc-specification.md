# API 资源文档规范

## 资源描述
描述当前的资源，便于使用者了解资源。

## 输入参数说明
在表格中描述请求参数，可使用字段：字段、类型、必选、描述；
如果希望使用其它字段名，可以根据`markdown标准语法`或者工具栏的`表格`按钮，添加适合自己的行和列。

## 调用示例
给出请求参数的示例。

## 返回结果
资源返回的结果说明，根据实际情况填写返回字段。

## 返回结果说明
根据实际输出，填写相应的表格项，现阶段有这三项: 字段、类型、描述
